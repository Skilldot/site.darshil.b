What’s Included in Each Service?

  • Static Website (1-page): A clean, responsive single-page website designed using HTML, CSS, and optionally JavaScript. Ideal for personal profiles, portfolios, or small business overviews. Includes custom styling, mobile compatibility, a contact form (if needed), and GitHub Pages hosting.
  • Static Website (multi-page): A full static website with multiple linked pages (e.g., Home, About, Services, Contact). Consistent design across pages, basic responsive layout, clean navigation, and GitHub Pages hosting. Good for personal brands, small businesses, or informational websites.
  • Blog Setup & Styling: I help you build a simple blog layout using static tools or raw HTML/CSS. This includes a homepage that links to blog posts, consistent post layout styling, custom fonts and color themes, and GitHub Pages deployment. Great for writers and learners.
  • Landing Page: A visually focused, scrollable one-page website designed to highlight a product, event, or service. Includes structured sections like headline, features, testimonials, call-to-action (CTA) buttons, and contact info. Mobile responsive and styled for clarity.
  • Content Writing: Clear, original writing for blogs, web pages, or informational posts. I can tailor the tone (casual, formal, instructional, etc.) to suit the target audience. Includes formatting for readability, basic SEO structuring (headings, keyword focus), and revisions to match your voice.
  • Additional Revisions: While basic revisions are free, this covers major layout changes, rewriting entire sections, or large design/style overhauls after initial delivery. This ensures flexibility while respecting both time and scope.
